# Plotting
Marconi.jl uses Plots.jl as the plotting framework with (currently) InspectDR as
the plotting backend.

Currently, one can plot networks on
* Smith Charts
* Rectangular Plots
* Polar Plots

* Circles on Smith Charts
  * Gain Circles
  * Q Circles
  * VSWR Circles
  * Constant-R Circles
  * Constant-G Circles
  * Constant-X Circles
  * Constant-Y Circles
