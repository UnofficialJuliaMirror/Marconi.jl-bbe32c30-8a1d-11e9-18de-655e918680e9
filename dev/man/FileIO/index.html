    File IO

    File IO

    Marconi.jl supports reading and writing standard Touchstone files with most of Touchstone spec implemented including n-ports, non standard port impedances, and S, Y, Z, H, and G parameter reading.

    Marconi.jl currently does not support noise parameters nor per-port impedance.
